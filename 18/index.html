🛠️ 도전! 엔트리로 감정 분석 AI DJ 만들기

엔트리에서 감정 4개를 직접 학습시켜 노래를 추천받아요. 아래 파일을 내려받아 열고, 클래스마다 예시 문장을 붙여넣어요.

엔트리 파일 내려받기 (감정에 따른 노래추천.ent) ↓
  1. 위 버튼으로 .ent 파일을 내려받고, 엔트리(playentry.org)에서 [작품 불러오기]로 열어요.
  2. 인공지능 블록의 학습하기에서 감정 클래스 네 개가 있는지 확인해요: 기쁨/신남, 슬픔/우울, 화남/스트레스, 불안/긴장.
  3. 아래 카드의 [예시 문장 복사]를 눌러, 같은 이름의 클래스 입력칸에 붙여넣기 해요. (한 줄에 한 문장씩 들어가요.)
  4. [학습하기]를 눌러요. AI는 마음을 읽는 게 아니라 표현의 패턴을 익히는 거예요.
  5. 학습에 안 쓴 새 문장을 넣어 결과를 확인하고, 추천 음악이 어울리는지 내가 판단해요.
