# 에셋 생성 프롬프트 (생성형 AI용)

만들어서 `assets/` 폴더에 **아래 파일명 그대로** 넣으면 앱이 자동으로 인식합니다.
(파일이 없으면 이모지로 표시되므로, 하나씩 만들어 넣어도 그때그때 적용됩니다.)

## 공통 스타일 (모든 프롬프트 앞에 붙이기)

> 초등학교 교재용 플랫 벡터 일러스트 스타일, 밝은 파스텔 톤에 스카이블루(#5B9CF6)와 민트(#34D399) 포인트, 두꺼운 외곽선 없이 부드러운 그림자, 흰색 또는 투명 배경, 정사각형 구도, 텍스트 없음

모두 "동그란 메달(배지) 안에 아이콘" 구도로 통일합니다.

## 넣는 방법

1. 생성한 이미지를 위 파일명으로 저장 (PNG, 투명 배경 권장)
2. `assets/` 폴더에 복사
3. 브라우저 새로 고침 — 자동 적용 (없는 파일은 이모지 그대로)
