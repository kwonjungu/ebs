배지 퀴즈 6관문(분류·예측·군집화·생성형 AI 확인·안전·윤리·사람 확인) → 나만의 AI 서비스 설계(포트폴리오 6칸) + 안전 방패 2개 이상 → AI 탐험가 다짐·서명 → 라이선스 카드 발급(도장·색종이) → 우리 반 갤러리
