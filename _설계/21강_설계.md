# 21강 체험 웹 설계서 — AI가 쓴 글, 편집장이 되어 고쳐 보기 (AI 글 편집장)

- 경로: `/21/index.html` (단일 파일, 바닐라 JS, 서버·API 키 없음)
- 포인트 컬러: **sky** (`--sky: #5B9CF6`, `--sky-light: #EBF3FF`)
- 콘셉트: **편집장 게임**. AI가 쓴 체육대회 기사 초안(교재 소재)에서 문제 문장을 찾아 클릭 → 유형 판정 → 실제 기록 카드 대조 → 고친 문장 선택 → 초안 vs 편집 후 비교. "**확인할 수 있는 사실(순위·금액)부터**"를 게임 규칙으로 강제한다.
- 핵심 메시지: "AI 글은 완성품이 아니라 초안. 순위·금액·날짜는 기록과 대조, 표현은 목적·독자에 맞게, 최종 판단은 사람(편집장)."

---

## 1. 학습 목표 연결

| 교재 학습 목표 | 체험 웹 대응 |
|---|---|
| ① AI가 쓴 글이 왜 '초안'인지 이해 | 도입에서 AI 초안 기사가 타자 연출로 좍 나옴 → "멋진데… 그대로 실어도 될까?" 문제 제기 |
| ② 사실 확인 부분과 표현 다듬기 부분 구분 | 문장 클릭 시 [사실 확인 필요]/[표현이 과함] 2버튼 유형 판정 |
| ③ 과장·어색한 문장을 기사답게 고치기 | 고친 문장 3지선다(과장 유지/과잉 수정/알맞음) 선택 |
| ④ 확인·다듬는 편집 태도 | '사실 먼저' 잠금 규칙 + 편집 완성도 게이지 + 최종 비교·체크리스트 |

## 2. 단계 구성 (진행 바 단계명)

```
① 준비  →  ② 초안 살펴보기  →  ③ 편집장 미션  →  ④ 확인하고 배지 받기
```
- ①: 안전 약속 3개 체크(가상의 이야기·초안 태도·최종 판단은 사람).
- ②: 개념 체험. AI 초안 등장 → 기록 카드 공개 → "다른 곳 찾기" 워밍업(자유 클릭, 힌트만).
- ③: 미션. 편집 규칙 발동 — 1라운드 사실 확인(2문장) → 2라운드 표현 다듬기(3문장) → 최종 기사 비교.
- ④: 퀴즈 2문항 + 배지.

## 3. 화면·상호작용 상세

### ① 준비
- 헤더: 칩 "AI 탐험대 · 중급 21강", 제목 "📰 AI가 쓴 글, 편집장이 되어 고쳐요", 부제 "AI 초안에서 사실을 확인하고 표현을 다듬어요."
- 도입 카드: "마루가 AI에게 체육대회 기사를 맡겼어요. 멋져 보이는데… 교실이 술렁였어요. 오늘은 여러분이 편집장이에요!"
- 안전 약속 3개 체크 → [편집장 임명받기].

### ② 초안 살펴보기 (개념 체험)
1. **AI 초안 카드**: "🤖 AI가 쓴 초안" 헤더 + 기사 6문장이 타자 연출(문장당 ~0.5s)로 순서대로 나타남(제목: "6학년 2반, 가을 체육대회 우승!"). 연출 후 "멋지죠? 그런데…"
2. **실제 기록 카드**(amber 카드, 접었다 펼치는 토글): 교재 기록 5줄. "편집장의 무기는 **근거 자료**예요!"
3. **워밍업 — 수상한 문장 찾기**: 6문장이 각각 탭 가능한 블록(min-height 44px). 탭하면 힌트만 표시 — 문제 문장이면 "🔍 수상해요! ③에서 고쳐 봐요", 정상 문장이면 "✔ 기록과 맞아요". 몇 개를 찾았는지 카운터(수상한 문장 5개 중 n개 발견). 3개 이상 발견 시 [다음] 활성화.
4. 개념 노트: "AI는 문장을 매끄럽게 이어 쓰지만, **실제 기록을 확인하고 쓰는 건 아니에요.** 그래서 AI 글은 **초안**이에요."

### ③ 편집장 미션 (핵심)
**편집 규칙 배너**(상단 고정): "📌 편집장의 순서 — **1단계: 확인할 수 있는 사실(순위·금액·날짜)부터!** 2단계: 표현 다듬기"

- 기사 6문장 표시. 사실 확인형 문장(2개)은 sky 테두리 + 🔎 아이콘, 표현형 문장(3개)은 잠금 🔒 상태(회색). **표현형을 먼저 탭하면** 흔들림 애니메이션 + 코치 말풍선: "잠깐! 편집장은 **확인할 수 있는 사실부터** 봐요. 🔎 문장부터 고쳐요."
- 문장 탭 → **편집 패널**(카드 하단 슬라이드업)에서 3스텝 진행:

  **스텝 1 · 유형 판정**: "이 문장, 무엇이 문제일까요?" → [🔎 사실 확인 필요] / [🎈 표현이 과함] 2버튼. 오답이면 coral + 이유 한 줄, 재시도 가능(예: "'1등'은 기록으로 확인할 수 있는 정보예요.").

  **스텝 2 · 기록 대조**(사실 확인형만): 기록 카드가 패널 안에 뜨고, 해당 기록 줄이 자동 하이라이트. "초안: 1등 ↔ 기록: **2등**" 대조 표시가 양쪽에 빨간/초록 밑줄로 붙음. (표현형은 스텝 2 대신 "사실은 맞아요. 하지만 학교 신문에 어울릴까요?" 안내)

  **스텝 3 · 고친 문장 고르기**: 3지선다. 정답 선택 시 기사 본문의 해당 문장이 취소선 → 새 문장으로 교체(mint 배경 플래시), 편집 완성도 게이지 +20%. 오답 시 이유 표시 후 재선택(예: 과잉 수정 선택지 → "사실까지 지워 버리면 기사가 심심해져요. 기록에 있는 내용은 살려요!").

- 사실 확인형 2문장 완료 → 🔒 해제 연출("사실 확인 완료! 이제 표현을 다듬어요") → 표현형 3문장 진행.
- 5문장 모두 편집 완료 → **최종 비교 카드**: 좌 "🤖 AI 초안" / 우 "📰 편집 후 기사"(모바일 세로 2단). 바뀐 문장 하이라이트. 아래 최종 체크리스트 3개(교재 그대로, 자동 ✔): "실제 기록과 맞나요? / 과장이 줄었나요? / 기사에 어울리나요?" → [편집 완료 도장 찍기] 버튼 → "채택! ✅" 도장 애니메이션 → [다음: 확인 문제].

### ④ 퀴즈 + 배지
7절 참고. 2문항 정답 시 [배지 받기].

## 4. 파이프라인/시각화 설계

**(a) 편집 파이프라인**(15강 `.pipe/.pnode` 재사용, ③ 상단에 상시 표시·진행에 따라 점등):
```
1. AI가 초안을 썼어요        → "입력한 말 → AI가 이어 쓴 초안"
2. 사실을 기록과 비교해요     → 🔎 2문장 (완료 수 표시: 0/2)
3. 표현을 다듬어요           → 🎈 3문장 (0/3)
4. 최종 기사 — 사람이 판단해요 → 편집 완료 도장
```
교재 흐름도(입력한 말→초안→기록 비교→표현 다듬기→최종 기사)를 게임 진행 상태와 1:1 대응시킴.

**(b) 편집 완성도 게이지**: 진행 바 아래 가로 게이지(0→100%, 문장당 20%). 신문이 점점 완성되는 메타포(게이지 끝에 📰).

**(c) 대조 시각화**: 스텝 2에서 초안 숫자(빨간 밑줄)와 기록 숫자(초록 밑줄)를 화살표(↔)로 연결 — "확인할 수 있는 정보는 이렇게 맞춰 봐요"를 그림으로.

## 5. 폴백·예외 처리

- **외부 의존 0**: 텍스트·CSS·JS만 사용. 카메라·마이크·CDN·이미지 파일 없음 → 폴백 모드 불필요.
- 타자 연출: `prefers-reduced-motion`이거나 [건너뛰기] 탭 시 전문 즉시 표시.
- 잠금 규칙 예외: 표현형 선탭은 오류가 아니라 학습 피드백(코치 말풍선)으로 처리. 같은 문장 재탭 시 완료된 문장이면 "이미 고쳤어요 ✔" 토스트.
- 정상 문장(1번 문장) 탭: ③에서도 "이 문장은 기록과 맞고 차분해요. 그대로 두어요!"—'다 고치는 게 편집이 아님'을 학습.
- 편집 패널 진행 중 다른 문장 탭 방지(패널 닫기 버튼 제공, 진행 상태는 문장 단위로 저장).
- 오답 무한 재시도 허용(감점 없음) — 초등 대상, 막힘 방지. 2회 오답 시 정답 힌트 강조.
- 새로고침 시 처음부터(저장 없음). 상태는 전부 메모리 — 개인정보·기록 저장 없음.
- 좁은 화면: 최종 비교 카드는 좌우 → 상하 스택, 문장 블록 터치 target 44px 이상.

  if phase=='fact' && s.type=='tone':

### 8-3. UI 컴포넌트·재사용

- 문장 블록: `<button class="sent">` (전체 폭, 좌측 정렬, 유형 아이콘 🔎/🔒/✔) — 15강 `.quiz-opt` 스타일 변형.
- 편집 패널: 카드 하단 고정 슬라이드업(`transform: translateY`), 스텝 1→3 내부 show/hide.
- 코치 말풍선: 마루 편집장 이모지(🧢📰) + 말풍선 카드, 3초 후 자동 사라짐(`aria-live="polite"`).
- 도장: CSS로 원형 도장(회전 -12deg, scale 1.4→1 bounce) "채택 ✅".
- 진행 바 `go(n)`·약속 체크·퀴즈 `ans()`·배지 `finish()`·`esc()`는 15강 코드 그대로 재사용.
- 텍스트 입력창 없음(선택형만) → 오탈자·개인정보 입력 이슈 원천 차단.
